## Step 2: Configure Environment

1. **Edit .env file** (created automatically):
```bash
# Open .env and update:
SECRET_KEY=<auto-generated>
DATABASE_URL=postgresql://user:pass@localhost/dbname  # Optional
GROQ_API_KEY=your-api-key  # Optional for translation
```

2. **For production database** (recommended):
```bash
# PostgreSQL
DATABASE_URL=postgresql://username:password@localhost:5432/document_extractor

# MySQL
DATABASE_URL=mysql+pymysql://username:password@localhost:3306/document_extractor
```

## Step 5: Production Setup (Optional)

### Set up HTTPS with Nginx

1. **Install Nginx**:
```bash
sudo apt-get install nginx
```

2. **Copy configuration**:
```bash
sudo cp nginx.conf /etc/nginx/sites-available/document-extractor
sudo ln -s /etc/nginx/sites-available/document-extractor /etc/nginx/sites-enabled/
```

3. **Get SSL certificate**:
```bash
sudo apt-get install certbot python3-certbot-nginx
sudo certbot --nginx -d yourdomain.com
```

4. **Restart Nginx**:
```bash
sudo systemctl restart nginx
```

### Create System Service (Linux)

```bash
sudo cp document-extractor.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable document-extractor
sudo systemctl start document-extractor
```

## Common Issues

### Port already in use
```bash
# Linux/Mac
sudo lsof -i :8000
sudo kill -9 <PID>

# Windows
netstat -ano | findstr :8000
taskkill /PID <PID> /F
```

### Database connection failed
```bash
# Check PostgreSQL is running
sudo systemctl status postgresql

# Test connection
psql -U username -d document_extractor
```

### Permission denied for uploads
```bash
chmod 755 static/uploads/
```
