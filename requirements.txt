# ============================================================
# Installation Instructions:
# ============================================================
# 1. Install all packages:
#    pip install -r requirements.txt
#
# 2. Install Tesseract-OCR (optional but recommended):
#    - Windows: Download from https://github.com/UB-Mannheim/tesseract/wiki
#    - macOS: brew install tesseract
#    - Linux: sudo apt-get install tesseract-ocr
#
# 3. Get Groq API key (for translation):
#    - Visit: https://console.groq.com/keys
#    - Create free account
#    - Generate API key
#    - Add to .env file: GROQ_API_KEY=your_key_here
#
# 4. First run downloads AI models (~500MB):
#    - TrOCR model: ~400MB
#    - EasyOCR model: ~100MB
#    - Models cached in ~/.cache/ folder
#
# ============================================================
# System Requirements:
# ============================================================
# - Python 3.8 or higher
# - 4GB RAM minimum (8GB recommended)
# - 2GB free disk space for models
# - Optional: GPU with CUDA for faster processing
#
# ============================================================
# Troubleshooting:
# ============================================================
# If torch installation fails:
#   pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
#
# If PaddleOCR fails on Windows:
#   pip install paddlepaddle -i https://mirror.baidu.com/pypi/simple
#
# If opencv fails:
#   pip install opencv-python-headless
#
# For GPU support (NVIDIA only):
#   pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
# ============================================================
